# Images Directory

This directory contains image assets for the portfolio website.

## Usage

- Place your profile photos, project screenshots, and other images here
- Use descriptive filenames (e.g., `project-healthcare-portal.jpg`)
- Optimize images before uploading for better performance
- Recommended formats: JPG for photos, PNG for graphics with transparency, WebP for modern browsers

## Placeholder Images

The site currently uses placeholder images from via.placeholder.com. Replace these with actual images in production.

## Image Optimization Tips

1. Compress images using tools like TinyPNG or ImageOptim
2. Use responsive images with appropriate sizes for different devices
3. Consider using WebP format with JPG/PNG fallbacks
4. Add descriptive alt text for accessibility
5. Use lazy loading for images below the fold
